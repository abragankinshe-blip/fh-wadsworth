⚡ How it works: Zapier watches your Gmail/Outlook inbox → sends email data to your webhook URL → a Netlify serverless function saves it to JSONBin → your dashboard auto-syncs and the order appears instantly.
Step 1 — JSONBin.io (Free Cloud Storage)

Orders created by Zapier are stored here so they appear on all devices in real time. Free at jsonbin.io.

jsonbin.io → Account → API Keys → Master Key
Click Auto-create to generate a new bin, or paste an existing ID.
Step 2 — Netlify Environment Variables

In Netlify → Site Configuration → Environment Variables, add:

VariableValue
JSONBIN_KEYYour JSONBin Master Key
JSONBIN_BINYour JSONBin Bin ID
WEBHOOK_SECRETAny secret password (optional, recommended)

After adding these, trigger a redeploy in Netlify so the function picks them up.

Step 3 — Your Webhook URL

Enter your Netlify domain to generate the URL to paste into Zapier:

https://your-site.netlify.app/.netlify/functions/zapier-webhook
Step 4 — Build the Zap
  1. Go to zapier.com → Create Zap
  2. Trigger: Gmail (or Outlook) → Event: New Email. Connect your inbox and add filters if needed (e.g. subject contains "order" or "bag")
  3. Action: App: Webhooks by Zapier → Event: POST
  4. Paste your Webhook URL from Step 3. Set Payload Type to JSON
  5. Map the data fields below, then publish your Zap
Field Name (in Zapier)Map to Gmail fieldNotes
from_nameFrom NameBecomes the Client name
from_emailFrom EmailFallback if no name
subjectSubjectOrder description
body_plainBody PlainSaved as Notes
dateDateDate email was received
assigned_toType manually: VJ / Angelo / JamieOr use Zapier Filter to route by email address

If you set a WEBHOOK_SECRET, add a Zapier header: X-Webhook-Secret = your secret.
